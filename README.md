# Como iniciar o app
- Instalar o node
- Instalar o expo:

  `npm install --global expo-cli`

- Instalar o yarn:

  `npm install --global yarn`

- Executar os comandos na raiz do projeto:
  
  `yarn`

  `expo start`

- Baixar o app expo Go no seu celular e escanear o QR Code ou tentar abrir pelo emulador android
